Informativa sulla Privacy

Ultimo aggiornamento: 2 Gennaio 2026

DigiShoppingCard è un'applicazione progettata con il principio della Privacy by Design. La tua privacy non è solo una funzione, è il cuore dell'applicazione.

Nessuna raccolta dati

DigiShoppingCard non raccoglie, non memorizza e non trasmette alcun dato personale a server esterni. Non è richiesta alcuna registrazione e non vengono creati profili utente.

Archiviazione Locale (Local-First)

Tutte le informazioni relative alle tue carte fedeltà (nomi dei negozi, numeri di barcodce, categorie) vengono salvate esclusivamente all'interno del tuo dispositivo tramite la tecnologia localStorage del browser.

Questi dati non lasciano mai il tuo dispositivo a meno che tu non decida esplicitamente di esportare un backup manuale.

Backup Cifrato

Se utilizzi la funzione di esportazione dati, l'applicazione offre la possibilità di proteggere il file con cifratura AES-256-GCM tramite una password scelta da te. La password e i dati cifrati vengono gestiti localmente e non sono accessibili allo sviluppatore.

Servizi di Terze Parti

  • OpenStreetMap: L'app utilizza le API di OpenStreetMap per calcolare la distanza dai negozi. Questo processo avviene localmente per suggerire la carta corretta in base alla posizione. Nessun dato identificativo o coordinata GPS viene inviato a server centrali in modo persistente.

Open Source

DigiShoppingCard è un progetto Open Source rilasciato sotto licenza GNU GPLv3. Il codice sorgente è pubblicamente verificabile su GitHub per garantire la massima trasparenza sulle pratiche di gestione dei dati.

Contatti

Per domande sull'informativa o sul funzionamento dell'app, puoi consultare il deposito GitHub ufficiale.

Privacy Policy

Last updated: January 2, 2026

DigiShoppingCard is an application designed following the Privacy by Design principle. Your privacy is not just a feature; it's the core of the app.

No Data Collection

DigiShoppingCard does not collect, store, or transmit any personal data to external servers. No registration is required, and no user profiles are created.

Local Storage (Local-First)

All information related to your loyalty cards (store names, barcode numbers, categories) is saved exclusively on your device using the browser's localStorage technology.

This data never leaves your device unless you explicitly choose to export a manual backup.

Encrypted Backup

If you use the data export function, the app offers the option to protect the file with AES-256-GCM encryption using a password of your choice. The password and encrypted data are handled locally and are not accessible to the developer.

Third-Party Services

  • OpenStreetMap: The app uses OpenStreetMap APIs to calculate distance from stores. This process happens locally to suggest the correct card based on your location. No identifying data or GPS coordinates are persistently sent to central servers.

Open Source

DigiShoppingCard is an Open Source project released under the GNU GPLv3 license. The source code is publicly verifiable on GitHub to ensure full transparency regarding data handling practices.

Contacts

For questions about this policy or how the app works, you can consult the official GitHub repository.
